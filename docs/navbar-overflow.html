Navbar Overflow Detection

Three detection methods. Resize each navbar to trigger overflow.

Check 1: Flex-Item Compression

flex-wrap: nowrap — items compress instead of wrapping. Detected when scrollWidth > offsetWidth on any flex child.

Check 3: Container scrollWidth Fallback
