Button Shimmer Experiments

← Back to components

Six loading-state treatments layered on top of .btn-orange-to-blue. Each card below shows the effect running continuously — click a button to pause / resume so you can compare against the idle state. After picking a favorite we'll promote it into css/components.src.css as a real utility.

— if you've paused some buttons and want to compare them all in sync.

1. Sheen sweep

A soft white highlight slides diagonally across the button. Base gradient is untouched; the effect is purely an overlay. Classic skeleton-loader idiom.

click to toggle 
.btn-shimmer-sheen::after {
  background: linear-gradient(110deg,
    transparent 30%,
    rgba(255,255,255,.55) 50%,
    transparent 70%);
  transform: translateX(-100%);
  animation: sheen 1.6s ease-in-out infinite;
}
@keyframes sheen { to { transform: translateX(100%); } }

2. Angle rotation

The gradient's angle rotates a full 360° via an animated @property custom property. Same colors, the diagonal sweeps around. (Your suggested approach.)

click to toggle 
@property --angle { syntax: '<angle>';
  initial-value: 103deg; inherits: false; }

.btn-shimmer-rotate {
  background-image:
    linear-gradient(var(--angle), #e83155, #ffdfb9);
  animation: rotate 2.5s linear infinite;
}
@keyframes rotate { to { --angle: 463deg; } }

3. Position slide

Extends the gradient with repeated color stops, then slides background-position. The colors flow through the button.

click to toggle 
.btn-shimmer-slide {
  background-image: linear-gradient(103deg,
    #e83155, #ffdfb9, #e83155, #ffdfb9, #e83155);
  background-size: 250% 100%;
  animation: slide 2s linear infinite;
}
@keyframes slide { to { background-position: 250% 0; } }

4. Pulse

Gentle opacity fade. Minimal motion, low contrast — good for background sync affordances where the loading state shouldn't fight for attention.

click to toggle 
.btn-shimmer-pulse {
  animation: pulse 1.4s ease-in-out infinite;
}
@keyframes pulse {
  50% { opacity: 0.6; }
}

5. Conic spin

Swaps the linear gradient for a conic gradient that rotates around the button's center. Most overtly "spinner"-flavored — pairs naturally with a loading metaphor.

click to toggle 
@property --from { syntax: '<angle>';
  initial-value: 0deg; inherits: false; }

.btn-shimmer-conic {
  background: conic-gradient(from var(--from),
    #e83155, #ffdfb9, #e83155, #ffdfb9, #e83155);
  animation: conic 2.5s linear infinite;
}
@keyframes conic { to { --from: 360deg; } }

6. Sheen + pulse combo

Layers the sheen sweep on top of the pulse — together they read more like "actively working" than either alone. Slightly busier visually.

click to toggle 
.btn-shimmer-combo {
  animation: pulse 1.4s ease-in-out infinite;
}
.btn-shimmer-combo::after {
  /* same sheen as variant 1 */
}
